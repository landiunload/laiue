# Laiue World Format v1

Сохранение мира — каталог `saves/default` рядом с `laiue.exe`.
Запись происходит автоматически при выходе из игры и кнопкой
«Сохранить мир» (Настройки → Админ); загрузка — на старте, если
метаданные читаемы, иначе создаётся новый мир. Битые файлы никогда
не роняют игру: повреждённая часть просто игнорируется.

```text
saves/default/
  world.meta    версия формата, seed, время суток, версия игры
  chunks.dat    правки блоков (пишет и читает модуль world)
  player.dat    позиция, взгляд и режим игрока
  mods.lock     включённые моды и их версии на момент сохранения
  moddata/      по блобу на мод: <имя-пака>.bin
```

## world.meta

Построчный ASCII-текст, первая строка — сигнатура формата:

```text
LAIUE WORLD 1
seed = 42
time_minutes = 618
game = 0.5
```

`time_minutes` — игровое время суток (0..1439). `game` — версия игры,
записавшей сохранение (информационно).

## chunks.dat

Бинарный файл, little-endian:

| Поле | Тип |
|---|---|
| magic | `u32` = 0x3143574C (`LWC1`) |
| version | `u16` = 1 |
| reserved | `u16` |
| seed | `i64` |
| blockOrigin | 3 × InfiniteCoord — начало координат мира |
| chunkCount | `u32` |
| далее чанки | absolute (3 × InfiniteCoord), deltaCount `u32`, deltas `u32[]` |

InfiniteCoord сериализуется как `i32 sign`, `u32 limbCount`,
`u64 limbs[limbCount]` — координаты произвольной точности, формат
не теряет данных даже после телепортов `T` (возведение X в квадрат).
Дельта — упакованный блок: 18 бит индекса внутри чанка 64³ + 8 бит типа.

Загрузка сверяет seed и восстанавливает начало координат через
`WorldRebase`. **Ограничения v1:** начало координат должно быть
представимо в int64 (иначе сохранение не загружается), а правки чанков,
чьи абсолюты не представимы относительно восстановленного начала,
пропускаются. И то и другое достижимо только цепочками телепортов `T`;
файл при этом остаётся полным — v2 сможет прочитать всё.

## player.dat

Бинарная запись фиксированного размера: magic `LWP1`, версия,
позиция глаз `double[3]` (в локальных координатах восстановленного
начала мира), `yaw`/`pitch` `f32`, режим (0 — полёт, 1 — ходьба).

## mods.lock

UTF-8, по строке на включённый мод в порядке `mods/enabled.txt`:
`<имя-пака> <версия>`. При загрузке текущий состав сравнивается с
сохранённым; расхождение — предупреждение в отладочный вывод
(загрузка не блокируется).

## moddata/

Каждый мод хранит один блоб `<имя-пака>.bin` через API
`writeModData`/`readModData` (см. [modding.md](modding.md)): чтение —
в `LaiueModInit`, запись — при изменениях или в `LaiueModShutdown`,
который вызывается до записи сохранения при выходе.

## Версионирование

Номер в `LAIUE WORLD 1` и `version` бинарных файлов растут только при
несовместимых изменениях; совместимые расширения добавляют поля в хвост.
Незнакомая версия файла игнорируется (новый мир вместо порчи данных).
