# mods

Моды laiue — только нативные DLL-паки `<имя>.lmp`. Встроенные примеры
собираются из `sdk/examples/` целями CMake и попадают в runtime-каталог
`mods/` автоматически. Заголовок API и исходники примеров находятся в
runtime-каталоге `sdk/`; полное описание в исходном дереве —
`docs/modding.md`.
